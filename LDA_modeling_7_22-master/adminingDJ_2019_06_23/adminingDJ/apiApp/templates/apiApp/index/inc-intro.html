Introduction

API documentation for Agriculture project.

Allowed HTTPs requests:
GET
Get a resource or list of resources.
POST
Update a resource.
Usual Server Responses:
200
OK
The request was successful
400
Bad Request
The request could not be understood or was missing required parameters.
401
Unauthorized
Authentication failed or user doesn't have permissions for requested operation.
404
Not Found
resource was not found.
405
Method Not Allowed
Requested method is not supported for resource.
500
Internal Server Error
Unexpected condition was encountered.
